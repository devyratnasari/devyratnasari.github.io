AI Attendance

This is a group Capstone project II at Durham College with team members: Devy Ratnasari and Priyanka Singh. The purpose of this project is to create an Attendance system that utilise computer vision. The project conducted from May to August 2022.

Problems and Solutions

Problem statement:

  • Manual attendance is tedious and time-consuming
  • Additional administrative tasks for teachers
  • Vulnerable to proxies and manual errors

Proposed Solution:

  • AI Powered attendance software
  • Utilise computer vision and face recognition library to mark the attendance
  • System that recognise the faces and verify the presence or absence of each student
  • Data will be stored on the cloud

Features and Functions

  • Tackle Proxy Attendance
    Decrease the chance of student taking attendance for someone else.
  • Technology Powered
    AI and Cloud Based
  • Eliminate Human Error
    Built on face recognition algorithms that reduce the chance of human error
  • Increases Productivity
