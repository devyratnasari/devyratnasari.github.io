About Me

Hi, my name is Devy and I hold a Bachelor's degree in Computer Science from BINUS University, Indonesia, and Graduate Certificate in Artificial Intelligence Analysis, Design & Implementation from Durham College, Ontario. I worked as Software Developer in Indonesia for 2 years before I moved to Australia for Working Holiday.

During my stay in Australia, I was searching for life experiences and what I want to do with my life. Which made me tried different types of work. The pandemic's lockdown made me realise how much I loved to solving problems and simplifying things.
Later on, I found out that Machine Learning, Deep Learning is growing rapidly in the current industry. Which triggered me to learn more about that particular subject and I am glad that I did.
In my current study, I learned on how to preparing data, visualise and telling the story about the data which could help the company to make an important decision. As well as, how to build a model and develop an AI based web application which way much better compare to the normal application without Artificial Intelligence. My latest project is creating a web application for AI based attendance system that utilised computer vision and facial recognition. Here are some technologies I have been working with recently:
  • Python
  • Tensorflow
  • Scikit-Learn
  • TFOD2
  • Face Detection
  • Word2Vec
  • Flask API
  • Azure
  • Heroku
  • AWS EC2
