About Me

Hi, I’m Devy, a software developer and data scientist. I hold a Bachelor’s Degree in Computer Science from Bina Nusantara (BINUS) University, Indonesia and a Graduate Certificate in Artificial Intelligence Analysis, Design & Implementation from Durham College, Ontario, Canada.
I have previously worked was a software developer in Indonesia for 2 years before moving to Australia for a working holiday. During this time, I have also been able to work in a wide range of industries and roles which have provided many new life skills and experiences.
During the Covid pandemic lockdown period, I discovered a passion for solving problems and simplifying things which led me to find out that Machine Learning and Deep Learning is rapidly growing in the current industry. As a result of learning more about this industry, I have pursued study in the field and have learnt new skills and attained new knowledge on how to prepare data, visualise and explain how data can help companies to make important and informed decisions. I have also learnt how to build models and develop an AI based web application as opposed to the normal applications without AI.
My latest project has involved creating a web application for an AI based attendance system that utilises computer vision and facial recognition.
I also have experience in the following technologies which I have worked with:
  • Python
  • Tensorflow
  • Scikit-Learn
  • TFOD2
  • Face Detection
  • Word2Vec
  • Flask API
  • Azure
  • Heroku
  • AWS EC2
